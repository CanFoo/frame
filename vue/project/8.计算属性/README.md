## vue + vue-router + vue-loader + webpack

###概述：
学习笔记

###项目运行：
克隆项目

```
> $ git clone https://github.com/CanFoo/react.git
```

进入主目录下安装依赖包

```
> $ npm install
```

在主目录下启动webpack服务

```
> $ npm run dev
```

发布项目文件命令

```
> $ npm run build
```

执行完`npm run dev`命令后，打开浏览器 `http://localhost:8080/`运行项目
